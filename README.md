# Netflix User Behavior & Revenue Analysis (SQL)


## Project Overview
This project involves a comprehensive analysis of a Netflix user dataset containing 50,000 records. Using MySQL, I extracted key business metrics to understand revenue drivers and user demographics.

## Key Insights Found:
* **Total Users:** 50,000 users analyzed globally.
* **Top Revenue Plan:** The **Standard Plan** generates the highest monthly revenue (~$245,836).
* **Average Audience:** The typical viewer is approximately **41 years old**.
* **Popular Content:** **Comedy** and **Thriller** are the most-watched genres.
* **Market Leader:** The **USA** leads in Premium subscriptions and content completion rates (65%).

## SQL Skills Demonstrated:
* Data Aggregation (SUM, COUNT, AVG)
* Grouping and Sorting (GROUP BY, ORDER BY)
* Data Formatting (ROUND)
* Performance Filtering (WHERE, LIMIT)
